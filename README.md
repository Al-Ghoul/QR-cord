# QR-cord
QR-cord is a discord bot that generates QR codes on demand.

## Installation
1. Clone the repository:
```bash
    git clone https://github.com/qr-cord/qr-cord.git
```
2. To install dependencies:

```bash
bun install
```

3. To run:

```bash
bun run index.ts
```

## Technologies Used
- Bun
- TypeScript
- Discord.js
